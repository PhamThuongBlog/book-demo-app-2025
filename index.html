Book Demo Project

An Easy-to-Use Book Management Web App

About the Project

Book Demo là một ứng dụng web được thiết kế để minh họa cách sử dụng Git và GitHub để thêm, sửa, xóa sáchsách.

Các công nghệ sử dụng:

  • Frontend: HTML, CSS, JavaScript (optional)
  • Backend: Java Spring Boot
  • Database: MySQL
  • Deployment: Ansible, Terraform, AWS EC2

Goals

This project is created for learning and demonstration purposes. It helps developers understand full-stack development and DevOps automation.
